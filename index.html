What are Hyperlinks?

In simple words, a Hyperlink is a digital reference to data that user can follow or be guided to by clicking or tapping. In the context of web, hyperlinks are embeded in text, images, icons or other elements within web pages. A hyperlink points to a whole document or to a specific element within a document. A simple and common hyperlink embeded in text can be:

Click me to Google!

They allow users to navigate between different web pages and resources seamlessly, forming the interconnected network we use every day.

Different types of Hyperlinks

Text Hyperlink
A Hyperlink can be embeded in text, this is quite commonly seen in web pages. And text with Hyperlink is called Hypertext.
Visit Pokemon Web
HTML code:
<a href="https://www.pokemon.com/au/">Visit Pokemon Web</a>
Image Hyperlink
Hyperlinks can also be applied to images. Clicking on the linked image navigates the user to the designated URL. It is similar to text Hyperlink with only the text being changed to a image. Image Hyperlink can also become complicated. There is one type called image map Hyperlink. In a single image, we can draw multiple clickable areas linking to different destinations / documents.
HTML code:
<a href="https://www.pokemon.com/au/"><img src="pokemon.png" ></a>
Mailto Hyperlink
Another type is email Hyperlink. Hyperlink is embedded into e-mail address and allows visitors to send an e-mail message to this e-mail address.
Send Me An Email!
HTML code:
<a href="mailto:23870387@student.uwa.edu.au">Send Me An Email!</a>
Download Hyperlink
Hyperlink can also let user to initiate the action of downloading a file. This is achieved by download Hyperlink. The target resources can be documents, images, audios, videos and other resources hosted by a server.
Capture a wild Pikachu!
HTML code:
<a href="pokemon.png" download>Capture a wild Pikachu!</a>
Download Hyperlink
Hyperlinks can also be applied to images. Clicking on the linked image navigates the user to the designated URL.
Get a Pikachu!
HTML code:
<img src="pokemon.png" alt="Pokemon Image">
Bookmark Hyperlink
A bookmark Hyperlink, also known as an anchor Hyperlink, is a type of hyperlink that allow users to link to a specific section within a webpage. The destination doesn't have to the same page.
Go back to definition
HTML code:
<img src="pokemon.png" alt="Pokemon Image">
AJAX Hyperlink
When combining AJAX with hyperlinks, you can use JavaScript to intercept the click event on the hyperlink, perform an AJAX request to load new content, and then update the page accordingly without navigating to a new URL. This approach is often used to create single-page applications (SPAs) where the user experience feels more fluid and responsive.
Load list of capitals in Asia by clicking...
HTML code:
<img src="pokemon.png" alt="Pokemon Image">

How Hyperlink works in HTML

In web browsing, hyperlinks are typically visually distinguished, often appearing in a different color, font, or style, and may include symbols indicating the link target or document type. This styling, known as link decoration, is controlled using CSS. Additionally, browsers may change the mouse cursor to a hand icon to indicate a clickable link. By default, links are usually displayed in underlined blue text for unvisited links and underlined purple text for visited links. When a user clicks on a link, the browser opens the link's target, which may trigger another program to handle non-HTML file types depending on browser settings and plugins.

When the use click the link, the browser display the link's target. But the effect may vary with the hyperlink system or according to how the link is implemented. On WWW most hyperlinks cause the target document to replace the source page. But some are mark to cause the target to open in a new window.

Another behavior could be...the user clicks the link and then a AJAX call will get information from remote document Or API. In this case, the source document will not get refreshed entirely but part of the content get updated. This kind of link is called AJAX link. It appears as a link and can trigger part of the web page get updated instead of loading an entire page. However, the technical implementation really has nothing to do with the link itself. The link can be replace with most other elements such as a button, a text. It is just a dom element with a click event listener. The dom element does not has to be a hyperlink, it can be any dom with a click event listener.

More than HTML

Actually, Hyperlink is a broader concept. It is used in various technologies, not just limited to HTML.

  • XLink, as recommended by the W3C, introduces hyperlinks with enhanced capabilities beyond those found in HTML. These advanced links enable multidirectional connections and facilitate linking both within and between XML documents. They also encompass simple links, which, being unidirectional, do not provide additional functionality compared to standard HTML hyperlinks.
  • In virtual worlds, hyperlinks serve as gateways to navigate between diffrerent virtual locations, objects, or experience within the virtual environment. These hyperlinks can be embedded within the virtual world's interface or content, allowing users to seamlessly transition between various areas or interact with specific elements.
